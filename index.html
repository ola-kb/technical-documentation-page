Introduction

CSS is the language we use to style a Web page, to style and present HTML. CSS is a way, that your web pages look good.

CSS What it is

CSS - Cascading Styles Sheets

  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files
CSS selected selectors

A CSS selector selects the HTML element(s) you want to style. Letâ€™s look at three different ways you can select HTML elements.

  • ELEMENT
  • CLASS
  • ID

The first way to select an HTML element selector is by simply using the name, which is what we did above.

p {font-size: 15px;}

The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name.Each element can have multiple classes, and each class can also be applied to multiple elements as well.

.triangle {width: 200px;}

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character, followed by the id of the element.

#green-eyes {color: green;}

CSS colors

In CSS, a color can be specified by using few ways. It making our HTML elements look pretty. The easiest way is use predefined color names, for example: Orange, Black, White,LightGray.

We can use color for text or background. Then we have to use different code.

CSS Text Color:

p {color: white;}

CSS Background Color:

body {background-color: blue;}

In CSS, colors can also be specified using another values:

  • RGB
  • HEX
  • HSL

And now it's time for example. I'll show how to code the same color but in different ways.

An RGB color value represents RED, GREEN, and BLUE light sources.

title {color: rgb(0, 255, 0)}

A HEX color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.

title {color: #00ff00;}

In CSS, a color can be specified using hue, saturation, and lightness (HSL). Hue is a degree on the color wheel from 0 to 360. Saturation and lightness is a percentage value:

title {color: hsl(120, 100%, 50%;}

CSS fonts

Using a font that is easy to read is important. The font adds value to your text. It is also important to choose the correct color and text size for the font. In CSS there are five generic font families: Serif, Sans-serif, Monospace fonts, Cursive fonts and Fantasy.Each of them has an impact on the reception of the text (elegance, handwriting or mechanical writing, etc.)

body {font-family: Courier, monospace;}
